# Правовой статус RegioBuild (заготовка для пилота)

**Продукт:** справочный Telegram-помощник по фрагментам нормативных актов РФ (РНГП/ТСН регионов волны 1, СП 42, кураторские выдержки 123-ФЗ и СанПиН) для задач **коммерческого размещения**.

## Что это не является

- Не юридическая консультация и не адвокатская услуга.
- Не заключение о допустимости размещения объекта на конкретном земельном участке.
- Не замена проектной документации, экспертизы, согласований и работы юриста.

## Ответственность

Пользователь самостоятельно сверяет указанные пункты в первоисточнике и несёт ответственность за решения, принятые на основе ответов помощника. Разработчик не отвечает за убытки от использования справочных ответов без профессиональной проверки.

Муниципальные ПЗЗ, местные НГП и иные акты, не входящие в индекс, могут содержать более жёсткие требования.

## Техническая проверяемость

Ответы формируются на основе retrieval по индексу НПА. Упоминания пунктов программно сверяются с найденными фрагментами; при отсутствии проверяемой опоры система отказывается генерировать выдуманные нормы. В ответах приводятся ссылки на `source_url` из конфигурации регионов.

## Следующие юридические шаги вне репозитория

- Пользовательское соглашение / оферта для платного пилота.
- Политика обработки ПДн (152-ФЗ), если собираются данные сверх Telegram user id.
- При продаже лицензий / инвестициях — оценка регистрации ПО в Роспатенте и аудит лицензий зависимостей/моделей под коммерческое использование.
